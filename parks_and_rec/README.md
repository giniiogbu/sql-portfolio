# Parks and Recreation SQL Project

## Overview
This project uses the Parks and Recreation sample database to practice and demonstrate SQL skills, including querying, filtering, aggregations, and joins.

The goal of this project is to answer practical business-style questions using SQL while keeping queries clean, readable, and well-documented.

## Dataset
- Fictional Parks and Recreation database
- Tables include employees, departments, salaries, and roles
- Used strictly for learning and portfolio purposes

## Skills Demonstrated
- SELECT statements
- WHERE filtering
- ORDER BY
- Aggregate functions (COUNT, SUM, AVG)
- GROUP BY and HAVING
- JOINs (INNER, LEFT)

## Tools
- MySQL
- VS Code
- SQLTools extension
